DIMENSÕES DAS FOTOS DA EQUIPE
==============================

O CSS define .team-member-photo com aspect-ratio: 4/5 e object-fit: cover.
Isso significa que as fotos são recortadas centralizadas se a proporção
divergir. Para evitar surpresas, prepare os arquivos em 4:5 (vertical).

OTIMIZAÇÃO RECOMENDADA
========================

Antes de subir:
  1. Redimensione para 600×750 px (largura × altura)
  2. Comprima com qualidade 80–85% (JPG) — alvo: 80–150 KB por arquivo
  3. Ferramentas: Squoosh.app (Google), TinyJPG, ImageOptim
